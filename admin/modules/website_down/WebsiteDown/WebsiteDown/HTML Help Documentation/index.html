A) HTML Structure - top

This theme is a fixed layout with two sections. All of the information within the main content is nested within a div with an id of "page-container". Two of the sections are nested within an unordered list with an id of "page". Here is the general structure.

If you would like to edit the color, font, or style of any elements in one of these columns, you would do the following:

	#page a {
		color: #someColor;
	}

If you find that your new style is not overriding, it is most likely because of a specificity problem. Scroll down in your CSS file and make sure that there isn't a similar style that has more weight.

I.E.

	#wrapper #page a {
		color: #someColor;
	}

So, to ensure that your new styles are applied, make sure that they carry enough "weight" and that there isn't a style lower in the CSS file that is being applied after yours.

B) CSS Files and Structure - top

Four CSS files are used in this theme. The first one is a generic reset file. Many browser interpret the default behavior of html elements differently. By using a general reset CSS file, we can work round this. This file also contains some general styling, such as anchor tag colors, font-sizes, etc. Keep in mind, that these values might be overridden somewhere else in the file.

ie.css and ie8.css are for fixing the problems that are caused by some earlier versions of Internet Explorer.

The fourth file (blue.css, green.css, red.css, yellow.css, purple.css) contains all of the specific stylings for the page. The file is separated into sections using:

	/* === Layout === */

	some code

	/* === Countdown === */
	
	some code
	
	/* === Contact Form === */
	
	some code
	
	/* === Validation === */
	
	some code
	
	/* === Newsletter === */
	
	some code
	
	/* === Social === */
	
	some code
	
	/* === Twitter === */
	
	some code
	
	etc, etc.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

If you would like to change the color scheme of the theme with the existing alternatives, you just have to change the name of the .css file under style.css. 

@import "style/css/green.css";

The example above uses green color scheme. If you would like to use blue, you should replace the code with:

@import "style/css/blue.css";

C) JavaScript - top

This theme imports six Javascript files.

  1. jQuery
  2. jCarousel
  3. Countdown
  4. Twitter
  5. Sliding Labels
  6. jQuery Inline Validation
  1. jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. jCarousel is used for the transition between sections. You can learn more about the script and other usage tips from here.
  3. jQuery Countdown is used for the countdown on first section. To change the time you should replace the following (YYYY, MM, DD) under custom.js: 
    j(function () {

		var newYear = new Date(); 
    	newYear = new Date(YYYY, MM - 1, DD);
    	j('.defaultCountdown').countdown({until: newYear}); 
  
		});
    For further usage tips of the script you can get help from here
  4. Twitter is used for displaying last twitter message on the site. To be able to display your own twitter messages you should replace the following under custom.js 
     getTwitters('twitter', {
        id: 'YOUR TWITTER ID', 
        count: 1, 
        enableLinks: true, 
        ignoreReplies: false,
        template: '%text% - %time%',
        newwindow: true
    });
    For further usage tips of the script you can get help from here
  5. Sliding Labels were used in the contact form to slide the labels (name, email, message). You can learn more about the script here
  6. jQuery inline validation is used for form validation which is used in the contact form on contact section. For further usage tips of the script you can get help from here

C) PHP / CONTACT and SUBSCRIPTION - top

  • To receive the emails that are sent via contact form, you must do the following:
    • Open the contact.php under the HTML theme folder (fleepy) that you previously uploaded.
    • On 11th line you will see the following: 
      $EmailTo = "yourname@domain.com";
      Replace the quoted sample address with your email address.
    • After the changes save the document.
  • Do the same with subscribe.php to receive the email addresses of the subscribers.
